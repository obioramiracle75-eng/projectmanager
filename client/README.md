TaskDuty - Task Management Application

Project Overview

TaskDuty is a full-stack task management application that allows users to create, view, edit, and delete tasks. The application helps users organize their activities by assigning tags such as Urgent, Important, and Personal.

Features

Frontend

- View all tasks
- Create new tasks
- Edit existing tasks
- Delete tasks
- Responsive user interface
- Task categorization using tags

Backend

- REST API built with Express.js
- MongoDB database integration
- CRUD operations for tasks
- Environment variable configuration

Technologies Used

Frontend

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS

Backend

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- dotenv

Known Issues Encountered During Development

- MongoDB Atlas connection errors caused by environment variable configuration.
- Incorrect ".env" file location prevented environment variables from loading.
- DNS resolution issues while connecting to MongoDB Atlas.
- Backend startup issues caused by incorrect TypeScript runtime configuration.
